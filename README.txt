README - كيف ترفع الصفحة لتحصل على رابط مباشر (سهل على الهاتف)

الملفات الموجودة في هذا الأرشيف:
- index.html  (صفحة الموقع)
- README.txt  (هذا الملف)
- profile.jpg  (صورة بروفايل: استبدلها بصورتك)
- bg.jpg       (خلفية الصورة: استبدلها بخلفيتك)
- weekend.mp3  (ملف الأغنية: استبدله بملف MP3 للأغنية)

خطوات سريعة لرفع الموقع والحصول على رابط مباشر (Netlify Drop) — أسهل طريقة من الهاتف:
1. فك الضغط عن الأرشيف على هاتفك (إذا لم تفعل).
2. افتح متصفح الهاتف وادخل إلى: https://app.netlify.com/drop
3. اسحب أو اختر مجلد يحتوي على الملفات (index.html وملفات الصور وweekend.mp3) — أو اسحب ZIP مباشرة.
4. سيفتح الموقع، انتظر دقائق وسيعطيك Netlify رابطًا عامًا مثل: https://wonderful-nightingale-12345.netlify.app
5. افتح الرابط وجرب الصفحة. إن لم تعمل الموسيقى تلقائيًا فاضغط زر التشغيل.

بدائل (إذا أردت تعليمات مفصّلة):
- GitHub Pages: أنشئ مستودع جديد واحمل الملفات عبر واجهة الويب ثم فعّل Pages (غالبًا داخل Settings > Pages).
- Firebase Hosting: يحتاج حساب Google وواجهة سطر أوامر (صعبة على الهاتف).

ملاحظات مهمة:
- استبدل profile.jpg وbg.jpg وweekend.mp3 بالملفات الحقيقية قبل الرفع.
- المتصفحات تمنع تشغيل الصوت تلقائيًا؛ ستحتاج الضغط على زر التشغيل.
- إن أردت، أستطيع تعديل index.html ليتضمّن روابط حقيقية أو أيقونات أخرى.

لو تريد، أضغط "تحميل" لتحصل على ملف ZIP جاهز بالملفات الافتراضية.
